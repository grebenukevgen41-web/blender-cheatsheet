Реалістичне скло (Cycles)
Матеріал
  1. 1Новий матеріал → Principled BSDF
  2. 2Transmission = 1.0
  3. 3IOR = 1.45 (звичайне скло)
  4. 4Roughness = 0 (чисте) або 0.05–0.1 (матове)
  5. 5Base Color = дуже темно-зелений або сірий тінт
  6. 6Render Engine = Cycles (Eevee потребує додаткових налаштувань)
Порада: В Eevee додатково: Render Properties → Screen Space Reflections (увімкни) + в матеріалі Blend Mode = Alpha Hashed

Матеріал тканини
Матеріал
  1. 1Principled BSDF → Base Color = тканинна текстура
  2. 2Roughness = 0.8–1.0 (тканина матова)
  3. 3Sheen = 0.5–1.0 (крайній блиск волокон)
  4. 4Підключи Normal Map (тканинна фактура)
  5. 5Subsurface = 0.05 (легке просвічування)
  6. 6Mapping: Scale 5–20x для дрібного тайлу
Де брати текстури: ambientcg.com, polyhaven.com — безкоштовні PBR текстури тканини

Штора (Cloth Simulation)
Симуляція
  1. 1Plane → S масштабуй до розміру штори → Subdivide 20-30x
  2. 2Виділи верхні вершини → Vertex Group "Pin" → Assign
  3. 3Physics Properties → Cloth → Preset: Silk або Cotton
  4. 4Cloth → Shape → Pin Group = "Pin"
  5. 5Відтворити симуляцію — штора падає. Зупини на кращому кадрі
  6. 6Apply modifier → застосувати результат

Паркет / Підлога
Матеріал
  1. 1Image Texture (дошки) → Mapping node → Scale X=5, Y=1 (довгі дошки)
  2. 2Roughness = 0.3–0.6 (лак) або 0.7–0.9 (матовий)
  3. 3Normal Map (деревна фактура) — підключи Non-Color!
  4. 4Для варіації: Noise Texture → додай до Roughness через MixRGB
  5. 5UV: "Follow Active Quads" для правильного напряму дощок
Результат: Lakований паркет з натуральною варіацією. polyhaven.com/textures → Wood

Бетон / Штукатурка
Матеріал
  1. 1Base Color = бетонна текстура (сірувата)
  2. 2Roughness Map (Non-Color) → Roughness вхід
  3. 3Normal Map (Non-Color) → Normal Map node → Normal
  4. 4Для варіації Roughness: Noise Texture (Scale=50) + MixRGB → Roughness
  5. 5Mapping: Scale 2–5 для правильного розміру
Без текстур: Principled: Base Color = RGB(0.4,0.4,0.4), Roughness = 0.9 + Bump з Noise Texture
🔩
Метал (матовий / глянець)
Матеріал
  1. 1Principled BSDF → Metallic = 1.0
  2. 2Base Color = колір металу (сталь = темно-сіра, золото = тепло-жовта)
  3. 3Roughness: 0 = дзеркальний | 0.3 = сатин | 0.7 = матовий
  4. 4Brushed steel: Anisotropic=0.7 + Noise Texture (Scale=200) → Anisotropic Rotation
  5. 5Normal Map (подряпини) → Normal для зносу
Швидко: Metallic=1, Roughness=0.3 → вже виглядає добре без текстур

LED стрічка / Неон
Матеріал
  1. 1Меш LED смуги → Principled BSDF → Emission Color = потрібний колір
  2. 2Emission Strength = 5–50 (чим більше — тим яскравіше)
  3. 3Eevee: Render Properties → Bloom → увімкнути → Intensity = 0.05
  4. 4Cycles: Emission матеріал реально освітлює сцену
  5. 5Окремо: додай Area Light поруч з той самий колір для чіткого освітлення
Хак: Light Path node → Is Camera Ray → MixShader: камера бачить яскраве, рендер освітлює реально

Освітлення інтер'єру
Освітлення
  1. 1World → HDRI (денне небо або студія) — дає м'яке заповнення
  2. 2У кожному вікні — Area Light → Light → Portal (тільки Cycles)
  3. 3Sun Light через вікно — Angle = 1-2°, Strength = 3–8
  4. 4Штучне світло: Area/Point/Spot з IES профілем або Emission mesh
  5. 5Samples: 256–512 + Denoise = достатньо
Загальна проблема: темно у кутах — збільш HDRI Strength або додай weak Fill Area light

Освітлення екстер'єру
Освітлення
  1. 1World → Environment Texture → HDRI неба (hdri-haven.com)
  2. 2Sun Light: Strength = 3–6, Angle = 0.5° (чіткі тіні)
  3. 3Поверни Sun і HDRI в одному напрямку (Mapping → Rotation Y)
  4. 4World Strength = 0.3–0.8 (підсвічування від неба)
  5. 5Cycles: 128-256 samples + Denoise. Render Time = 1-3 хв
Золота година: HDRI захід + Sun Strength=1, теплий колір #FFD580, Angle=2°

Розсіяти об'єкти (трава, камені)
Геометрія
  1. 1Виділи поверхню → Properties → Particles → +
  2. 2Type = Hair, Advanced = увімкнути
  3. 3Render → Render As = Object → Instance Object = твій об'єкт
  4. 4Number = кількість. Scale / Scale Randomness для варіації
  5. 5Rotation → рандомізуй для природнього вигляду
Краще: Geometry Nodes → Distribute Points on Faces → Instance on Points. Більш гнучко і швидко

Вода / Басейн
Матеріал
  1. 1Principled BSDF: Transmission=1, IOR=1.33, Roughness=0
  2. 2Base Color = дуже темно-синій або cyan (0.01, 0.05, 0.08)
  3. 3Хвилі: Modifier → Ocean (або Displacement з Wave Texture)
  4. 4Cycles only. Eevee = Screen Space Reflections + Alpha Hashed
  5. 5Caustics: Render Properties → Light Paths → Caustics (повільно але красиво)
Пул: Тонкий Plane з матеріалом Water + Ocean modifier зі Scale=0.01

Рослини / Листя (Alpha cutout)
Матеріал
  1. 1Principled BSDF: Base Color = текстура листа (PNG з альфа)
  2. 2Image Texture → Alpha вихід → Alpha вхід Principled
  3. 3Material Properties → Settings → Blend Mode = Alpha Clip
  4. 4Shadow Mode = Alpha Clip (для правильних тіней)
  5. 5Roughness = 0.7–0.9. Subsurface = 0.1 (просвічування листа)
Subsurface колір: зелений тінт для реалістичного просвічування сонячного світла
🤯 WTF — найчастіші проблеми максівця в Blender
Об'єкт чорний після Join або import
Перевернуті нормалі. Edit Mode → Alt+N → Recalculate Outside (Shift+N). Або Viewport Overlays → Face Orientation — червоне = перевернуто.
Boolean дає жахливу топологію
У Boolean Modifier переключи Solver = Exact (замість Fast). Ще: обидва меші повинні мати правильні нормалі і не перетинатися до boolean. Аддон Bool Tool (вбудований) полегшує роботу.
Ctrl+Y не робить Redo
В Blender Redo = Ctrl+Shift+Z. Ctrl+Y не працює. М'язова пам'ять з Max — найбільша засідка на першому тижні.
Скло рендериться чорним
1) Переконайся що Engine = Cycles (Eevee потребує окремих налаштувань). 2) Render Properties → Light Paths → Max Bounces: Transmission = мін 8. 3) В Eevee: Render Properties → Screen Space Reflections → увімкнути + Blend Mode в матеріалі = Alpha Hashed.
Текстура не відображається (суцільний колір)
Немає UV розгортки або неправильні координати. 1) Edit Mode → A → U → Smart UV Project (швидке рішення). 2) Або в нодах: Texture Coordinate → Generated → Mapping → Image Texture (без UV).
Modifier (Bevel, Solidify, Cloth) поводиться дивно / нерівномірно
Не застосований Scale. Ctrl+A → Apply All Transforms. Це як Reset XForm в Max — обов'язково після масштабування. Особливо важливо для Cloth і фізики.
Об'єкт зник, не можу знайти
1) Alt+H = показати все. 2) Відкрий Outliner — можливо в іншій Collection. 3) A щоб виділити все що видно. 4) Header Outliner → фільтр → увімкни All Visibility щоб бачити всі об'єкти.
Subdivision Surface деформує модель — стає "кашею"
Потрібно контролювати subdivision. Два способи: 1) Додай supporting loop cuts (Ctrl+R) поруч з гострими ребрами. 2) Виділи ребра → Shift+E → 1.0 (Crease = гостро без loop cuts). Другий спосіб краще для hardsurface.
Рендер шумний як сніг
1) Збільш Samples (Render Properties → Sampling → Render: спробуй 256-512). 2) Увімкни Denoise (OptiX якщо NVIDIA, OpenImageDenoise для CPU) — прибирає шум при меншій кількості samples. 3) Перевір що Source Light достатньо яскравий.
В Eevee немає тіней від маленьких об'єктів
Render Properties → Contact Shadows → увімкнути. Також перевір кожне світло → Light Properties → Shadow → увімкнено. Contact Shadows вирішують ближні дрібні тіні яких Eevee не вираховує за замовчуванням.
Натиснув ПКМ — і все скасувалось
В Blender ПКМ = Cancel операції (протилежно від Max!). Підтвердити = Enter або Left Click. Esc теж скасовує. М'язова пам'ять — тиждень і звикнеш.
Об'єкт обертається або масштабується не з центру
Невірно розміщений Origin (pivot). ПКМ → Set Origin → Origin to Geometry. Або переміщуй Origin окремо: Edit Mode → виділи потрібну вершину → Object → Set Origin → Origin to Selected.
FBX з Max імпортується в 100x більшому масштабі
При Export з Max: перевір Units = Centimeters і Apply Scale. При Import в Blender: FBX → Settings → Apply Transform → увімкнути. Або після імпорту: Ctrl+A → Apply All Transforms.
Modifier не можна застосувати (кнопка Apply сіра)
Ти в Edit Mode. Apply доступний тільки в Object Mode. Натисни Tab щоб вийти з Edit Mode → тоді Apply.
Кожного разу вручну підключати всі текстурні ноди?
Встанови аддон Node Wrangler (вбудований, тільки увімкни в Preferences → Add-ons). Виділи Principled BSDF → Ctrl+Shift+T → вибери всі файли текстур → сам підключить Base Color / Normal / Roughness / Metallic правильно.

  1. !В Blender Object — це контейнер (позиція, ротація, масштаб). Mesh — це геометрія всередині.
  2. !Один і той самий Mesh може бути у декількох об'єктах — зміна одного = зміна всіх (як Instance).
  3. !Це поясняє чому Alt+D (instance) ≠ Shift+D (копія з новим мешем).

  1. !Origin (помаранчева крапка) = точка відліку, обертання, масштабу.
  2. !Origin може бути де завгодно — і в геометрії, і за її межами.
  3. !Якщо об'єкт обертається "не з того місця" → ПКМ → Set Origin → Origin to Geometry.
